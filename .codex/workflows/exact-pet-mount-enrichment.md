# Exact pet/mount enrichment workflow

Use this when Codex is asked to add or use pet/mount facts.

1. Verify whether the record is only a drop-source stub.
2. Import exact pet/mount page fields before using stats, talents, cards, speed, or stat boosts.
3. Run `pet_mount_exact_readiness_report.py`.
4. Keep best-pet, hatch-value, and route-optimization claims locked unless a strategy-reviewed record exists.
5. For mount claims, distinguish movement speed from combat stat bonuses.
